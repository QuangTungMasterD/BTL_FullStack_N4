# Client - Course Management System
<!--  -->

Frontend application for the Course Management System built with:

* Vue 3
* Vite
* Vuetify
* Pinia
* Vue Router
* Axios

---

## Requirements

* Node.js >= 20
* npm >= 10

---

## Install dependencies

```bash
npm install
```

---

## Run development server

```bash
npm run dev
```

Default:

```txt
http://localhost:5173
```

---

## Build project

```bash
npm run build
```

---

## Preview production build

```bash
npm run preview
```
